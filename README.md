# MBI807B — Business Intelligence & Data Warehousing Project

This repository showcases an end-to-end Business Intelligence solution developed for MBI807B.

The project demonstrates:

- ETL process design
- Dimensional data warehouse modelling (Star Schema)
- SQL analytical layer development
- Power BI dashboard implementation
- Structured BI documentation

---

## Project Architecture

Raw Data  
→ ETL (Cleaning & Transformation)  
→ Data Warehouse (Fact + Dimensions)  
→ SQL Analytics Layer  
→ Power BI Dashboard Insights  

---

## Key Components

### Data Warehouse
- Fact table containing measurable metrics
- Dimension tables supporting segmentation and filtering
- Indexes applied for performance optimisation

### Analytical SQL Layer
- Aggregation queries
- Trend analysis
- Segmentation analysis
- Performance comparison queries

### ETL Workflow
- Data validation and cleaning
- Data type standardisation
- Key creation for relational joins
- Controlled loading into warehouse tables

### Power BI Reporting
- Executive summary KPIs
- Trend analysis visuals
- Segmentation dashboards
- Time intelligence measures (YoY analysis)

## Tools Used

- Python (ETL logic)
- SQL (Warehouse and analytical queries)
- Power BI (Data modelling & DAX)
